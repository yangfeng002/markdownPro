# 列表

## 1. 有序列表  
`描述：有序列表使用数字并加上 . 号来表示`  

`exp1:使用.`
1. 第一项  

2. 第二项  

3. 第三项  

 
## 2. 无序列表
`描述：无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记'`
 
`exp1:使用*`  
* 列表1  
* 列表2  

`exp2:使用+`  
+ list1  
+ list2  

`exp3:使用-`  
  - test1
  - test2



## 3. 列表嵌套
`描述：列表嵌套只需在子列表中的选项添加 "四个空格" 即可`
1. 第一项：
    + list1
	 + list1.1
	*list2  
2.第二项：
    *  第二项嵌套的第一个元素
	 *  第二项嵌套的第二个元素
      *  test

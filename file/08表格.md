## 1.表格
`描述：使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行`
`exp1:展示表格`  

|  序号  |  名称  |  
|  :--:   |  :--:   |  
|   1    | xp系统  |  
|   2    | windows系统  |  
|   2    | linux系统  |  
|   4    | unix系统  |

| 左对齐 | 右对齐 | 居中对齐 |  
| :----  | ---:    |  :--: |  
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
